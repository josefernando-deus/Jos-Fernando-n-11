Glossário de Termos de Desenvolvimento de Software

HTML (HyperText Markup Language)
É a Linguagem de Marcação de Hipertexto, utilizada para estruturar uma página web e seu conteúdo (como parágrafos, imagens e tabelas). É interpretada pelos navegadores.
Desenvolvendo (Desenvolvimento de Software)
O processo de criação, design, implementação e suporte de software. Envolve a escrita de instruções (código) que dizem a um computador o que fazer para criar programas, aplicativos ou websites.
JavaScript (JS)
Uma linguagem de programação que permite aos desenvolvedores criar páginas da Internet interativas, com conteúdo que se atualiza dinamicamente, controlando multimídias e animações.
Análise (Análise de Software ou de Sistemas)
A etapa do ciclo de vida do desenvolvimento de software preocupada com o levantamento e a compreensão dos requisitos e necessidades do cliente para que o software atenda às expectativas.
Código (Código-Fonte)
O conjunto de instruções escrito em uma linguagem de programação (como Java, Python ou JavaScript) que define um software e pode ser interpretado ou compilado por uma máquina para o funcionamento da aplicação.
Função (Programação)
Um bloco de código projetado para realizar uma tarefa específica. É um recurso essencial para a reutilização, organização e facilidade de manutenção do código (Princípio DRY - Don't Repeat Yourself).
Sistemas (Sistemas de Software)
Software que assegura o funcionamento do computador e permite que o usuário interaja com os componentes físicos e utilize diversos programas (Exemplo: Sistemas Operacionais).
Parâmetros (Programação)
Valores que são passados para uma função ou sub-rotina. A função utiliza esses valores para alterar ou definir o seu comportamento durante a execução.
Verificar (Verificação de Software)
No contexto de testes, é o conjunto de atividades que garante que o software implementa corretamente uma função específica, assegurando que o código está fazendo o que foi projetado para fazer.
Atualização (Atualização de Software)
O ato de mudar a versão de programas essenciais (como sistemas operacionais, drivers ou aplicativos) para o bom funcionamento, segurança e adição de novas funcionalidades.
Lógica (Lógica de Programação)
O conjunto de regras e técnicas utilizadas para projetar e desenvolver programas de computador. É a habilidade de pensar de forma estruturada e decompor um problema complexo em etapas simples (algoritmos).
Agrupar (Clusterização)
O ato de organizar ou aglomerar recursos de computação (como servidores ou dados) para garantir alto desempenho, distribuição de carga ou alta disponibilidade dos serviços.
